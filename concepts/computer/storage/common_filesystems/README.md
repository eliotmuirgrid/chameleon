# Common filesystems (index)

Hub: [Common filesystems (cross-platform)](../common_filesystems.md). Each file below is **one identity**—link here from OS or Git pages instead of stacking every format on one page.

## Pages

- [NTFS](ntfs.md) — usual default on Windows desktops; typically case-insensitive, case-preserving.
- [APFS](apfs.md) — modern Apple filesystem on Mac internal drives.
- [HFS+](hfs_plus.md) — older Apple filesystem, still common on externals.
- [ext4](ext4.md) — common native Linux layout; case-sensitive paths.
- [XFS](xfs.md) — journaled Linux filesystem; case like ext4 for ordinary paths.
- [Btrfs](btrfs.md) — advanced Linux filesystem; case like ext4 for ordinary paths.
