# Communication (concepts)

This folder holds **communication** ideas that apply **both to people and to AI**: how we give **feedback**, **clarity**, and **signals** so the other party can **learn**, **trust**, and **improve**. The same patterns show up in management, parenting, collaboration, and **prompting or evaluating** language models.
