# Singular path segments

## Concept

Use singular names for path segments.
A singular name usually maps to one clear page job.

Keep each segment as one clear token.
If you need extra words, use folder depth or detail pages.

## Why

- Clear ownership per page
- Lower naming ambiguity
- Better fit with [Concept - Detail Pages](../../../detail.md)
