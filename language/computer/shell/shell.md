## Team Workflow

We train one loop until it is automatic: one mission, one workflow, one shared toolset.

## Install For Every Teammate

1. Clone or update `~/chameleon`.
2. Add team tools to your shell `PATH`:
   - `export PATH="$HOME/chameleon/language/computer/shell:$PATH"`
3. Reload your shell:
   - `source ~/.zshrc`
4. Run commands from anywhere, and keep the same execution path every time.

## Why This Works (Theory of Constraints)

- Team flow is constrained more by inconsistent execution than by raw speed.
- Shared scripts enforce a repeatable loop under pressure.
- Every loop teaches the same moves, so handoffs stay clean.
- One improvement in the repo upgrades the constrained step for everyone.
- Consistency compounds: less rework, faster recovery, higher team tempo.
