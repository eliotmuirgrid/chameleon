# BAS documentation

This folder holds narrative documentation for the BAS package (base / shared runtime-style code): event loop, I/O helpers, containers, threading, and similar pieces. Use it for orientation and background, especially if you are new to how this module fits into the larger C++ tree.

**Related**

- [Shared make documentation](../../make/docs/index.md): how builds, dependencies, and `make` work in this repository.
- [Building C/C++ programs (basics)](../../make/docs/A.building-cpp-basics.md): compile versus link, objects, and executables, if those ideas are new.

The pattern is the same elsewhere: each package can keep deeper docs under `docs/` and a short `README.md` at the package root pointing here.
