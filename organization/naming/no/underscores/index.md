# No underscores in `concepts/` names

## Concept

Do not use underscores in file or folder names under `concepts/`.
Underscores often look like code identifiers (`snake_case`).
That makes concept paths look like API names instead of page names.

## What to do instead

Use one clear token per segment.
If a topic needs multiple words, use folder depth or a detail page.
