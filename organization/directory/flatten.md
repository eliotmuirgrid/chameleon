# Flatten deep paths

Use flat paths when a directory chain is too deep and each folder adds little value.

The goal is simple: keep meaning, remove extra layers.

How to flatten:

1. Start with the current path.
2. Find folders that only repeat or add weak detail.
3. Remove those folders.
4. Keep the same meaning in the new path or filename.
5. Stop when the path is short and still clear.

Good result:

- Fewer levels.
- Same meaning.
- Faster to scan, search, and maintain.
