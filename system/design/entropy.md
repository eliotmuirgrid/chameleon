# Entropy

**Entropy** (informal system use) is the tendency for things to drift toward disorder over time.

In plain words: if no one maintains a system, it usually gets messier.

## System view

- Small errors accumulate.
- Structure degrades.
- Signal gets buried in noise.
- The cost to find, decide, and act goes up.

## Core variables

- **Order level** - how clear and structured the system is.
- **Error rate** - how many small mistakes are added over time.
- **Maintenance effort** - how much active work pushes back on drift.
- **Recovery cost** - effort needed to restore clarity.

## In writing and language

When we simplify language recursively, meaning usually gets clearer at first.

Past a point, further simplification starts removing key meaning.
That is a practical entropy signal: information loss is rising faster than clarity gain.

## Quick test

After simplifying, ask:

1. Is the core meaning still intact?
2. Can the reader still make the right decision?
3. Did we remove precision that matters?

If "no" appears, entropy is increasing and the simplification should stop.
