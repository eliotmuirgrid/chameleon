# Algorithms (concepts)

Short notes on repeatable step-by-step methods for solving problems.

## Pages

- [Recursion](recursion.md) - solve a problem by solving a smaller version of the same problem until a stop rule is reached.
- [Binary search](binary/search.md) - find a target quickly in sorted data by repeatedly cutting the search space in half.
- [Iteration](iteration.md) - improve results through repeated passes using feedback and small adjustments.
